# 🧮 Python Calculator

A simple and powerful **CLI (Command Line Interface) Calculator** built using Python.
This project performs basic as well as advanced mathematical operations.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division (with zero handling)
* 🔼 Power (a^b)
* 🔢 Modulus (remainder)
* 🔁 Continuous calculation (loop system)

---

## 🧠 How It Works

1. User selects an operation
2. Enters two numbers
3. Program performs the operation
4. User can continue or exit

---

## ▶️ Usage

Run the program using:

```bash
python calculator.py
```

---

## 📌 Example

```
Enter operation: add
Enter number 1: 5
Enter number 2: 3
Output: 8
```
